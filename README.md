# Mocha & Crumb Café Website

A five-page website developed for the Web Development Fundamentals (WEDE5020W) module at Rosebank College.  
This project demonstrates the use of HTML, CSS, and JavaScript to create a professional, responsive website for a local café business.

---

## Project Overview
Mocha & Crumb Café is a modern, community-friendly coffee and cake shop that offers a warm atmosphere, handcrafted beverages, and freshly baked goods.  
The website serves as an online presence for the café, showcasing its menu, story, and contact details while allowing customers to make enquiries.

---

## Project Goals
- Apply HTML, CSS, and JavaScript principles to a practical web development project.  
- Design a simple, clean, and responsive website layout.  
- Implement an enquiry form with basic JavaScript validation.  
- Demonstrate folder structure, navigation links, and GitHub version control.

## Technologies Used
- HTML5 for structure  
- CSS3 for design and layout  
- JavaScript for interactivity and form validation  
- Git and GitHub for version control and hosting  
- Visual Studio Code as the primary development environment  

---

## Design Choices
- Colour palette: cream (#FFF7F0), latte (#F5E0D3), coffee (#4B2C20), mocha (#B97A56)  
- Font: Open Sans – clean and easy to read  
- Layout: consistent header and footer across all pages  
- Responsive design for different screen sizes  
- Compressed images for faster loading times  

---

## Form Validation
The enquiry form uses JavaScript to:
- Ensure all required fields are filled in  
- Validate email format  
- Ensure the pickup date is not in the past  
- Display confirmation or error messages to the user  

---

## How to Run Locally
1. Clone or download this repository.  
2. Open the folder in Visual Studio Code.  
3. Right-click on index.html and select "Open with Live Server" (or open it directly in a web browser).  
4. Navigate through the pages using the menu links.
