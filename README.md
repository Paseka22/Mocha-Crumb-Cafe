Project Overview

The Mocha & Crumb Café website is a multi-page responsive website designed for a café offering coffee, pastries, and light meals. The website was developed as part of the Web Development Essentials Practical Assessment. The project demonstrates foundational web development skills, including HTML structure, CSS styling, responsive design, multimedia integration, navigation, and form creation.

⸻

Website Goals and Objectives
	1.	Inform visitors about the café’s offerings, hours, and values.
	2.	Provide an engaging digital presence using images, branding, and a clean layout.
	3.	Ensure seamless navigation across all pages.
	4.	Implement a responsive layout for desktop, tablet, and mobile.
	5.	Allow customers to submit enquiries using a functional form.

⸻

Key Features
	•	Multi-page structure: Home, About, Menu, Enquiry, Contact
	•	Custom logo and consistent branding
	•	Responsive navigation bar
	•	Mobile-friendly layout using media queries
	•	Interactive enquiry form
	•	Images of menu items and café products
	•	Clean and accessible layout
	•	Fully functional GitHub repository
